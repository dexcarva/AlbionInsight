# Albion Insight (한국어)
Albion Insight는 Flet 프레임워크를 사용하여 Python으로 재구현된 Albion Online 게임용 크로스 플랫폼 (Linux, Windows, macOS) 통계 분석 도구입니다. 네트워크 트래픽을 분석하여 실버, 명성, 전투 데이터 (데미지 미터)를 포함한 실시간 게임 내 통계를 추적하도록 설계되었습니다.
## 기능
* **크로스 플랫폼 호환성:** Linux, Windows, macOS에서 기본적으로 실행됩니다.
* **실시간 추적:** Scapy 라이브러리를 사용하여 Albion Online 포트 (5055, 5056, 5058)에서 UDP 패킷을 스니핑합니다.
* **데미지 미터 구조:** 실시간 전투 통계 (가한 피해, 가한 치유, DPS)를 표시하기 위한 필요한 데이터 구조 및 UI를 포함합니다.
* **현대적인 UI:** Flet으로 구축되어 빠르고 네이티브처럼 보이는 데스크톱 애플리케이션을 제공합니다.
* **세션 관리:** 세션 통계 시작, 중지, 재설정 및 저장을 허용합니다.

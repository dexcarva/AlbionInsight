# Albion Insight - Analisis Statistik untuk Albion Online (Bahasa Indonesia)

Selamat datang di **Albion Insight**, port Python dari proyek AlbionOnline-Statistic.

Albion Insight adalah alat analisis statistik yang kuat dan lintas platform untuk game Albion Online. Kami bertujuan untuk menyediakan fungsionalitas yang kaya dari proyek asli, tetapi dengan kompatibilitas yang lebih luas dan basis kode yang modern.

## Fitur Utama

*   **Lintas Platform:** Berjalan di Linux, macOS, dan Windows (berbeda dengan proyek asli yang hanya mendukung Windows).
*   **Analisis Data Pasar:** Memantau dan menganalisis data rumah lelang.
*   **Pelacakan Aktivitas:** Log jarahan, meteran kerusakan, dan pelacakan ruang bawah tanah.
*   **Basis Kode Modern:** Dibangun dengan Python, memastikan pemeliharaan dan kontribusi yang lebih mudah.

## Instalasi

Untuk menginstal Albion Insight, ikuti langkah-langkah di bawah ini:

1.  **Kloning Repositori:**
    ```bash
    git clone https://github.com/dexcarva/AlbionInsight.git
    cd AlbionInsight
    ```

2.  **Instal Dependensi:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Jalankan Aplikasi:**
    ```bash
    python -m albion_insight
    ```

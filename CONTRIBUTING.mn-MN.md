## Ёс Зүйн Дүрэм

Энэхүү төсөл болон түүнд оролцож буй бүх хүмүүс манай Ёс Зүйн Дүрмээр зохицуулагдана. Оролцож байгаа тул та энэ дүрмийг дагаж мөрдөнө гэж хүлээгдэнэ. Хүлээн авалтгүй зан авирыг төслийн хянагчдад мэдээлнэ үү.

## Би Хэрхэн Хувь Нэмэр Оруулж Чадах Вэ?

### Алдаа Мэдээлэх

Алдаа мэдээлэл үүсгэхээс өмнө одоо байгаа асуудлуудыг шалгаж дахин давтагдахаас сэргийлнэ үү. Алдаа мэдээлэл үүсгэхдээ алдаа мэдээлэлийн загварыг ашиглан аль болох их дэлгэрэнгүй мэдээлэл оруулна уу.

**Сайн алдаа мэдээлэлд орно:**
- Тодорхой ба дүрслэлийн сайтай гарчиг
- Асуудлыг сайтар дахин үүсгэх алхамууд
- Хүлээгдэж байсан ба бодит үйлдэл
- Хэрэгтэй бол скриншотууд
- Таны орчны дэлгэрэнгүй мэдээлэл (ОС, Python хувилбар гэх мэт)
- Холбогдох логууд эсвэл алдаа мессежүүд

### Функцийн Санал Өгөх

Функцийн санал өгөх нь сайн байна! Функцийн хүсэлтийн загварыг ашиглан дараахыг өгнө үү:
- Функцийн тодорхой тодорхойлолт
- Энэ шийдэх асуудал
- Хэрэгжүүлэх боломжит аргууд
- Та авч үзсэн өөр хувилбарууд

### Код Хувь Нэмэр

Бид код хувь нэмрийг сайшаадаг! Эхлэх аргыг энд үзүүлэв:

1. **Репозиторийг Fork хийх** ба `master` салбараас өөрийн салбар үүсгэх
2. **Хөгжүүлэлтийн орчныг тохируулах** (доорх Хөгжүүлэлтийн Тохиргоо үзнэ үү)
3. **Өөрчлөлтөө хийх** манай кодчилолын стандартыг дагаж
4. **Өөрчлөлтөө сайтар шалгах**
5. **Баримт бичгийг шинэчлэх** шаардлагатай бол
6. **Pull Request илгээх** манай PR загварыг ашиглан

### Баримт Бичиг

Баримт бичгийн сайжруулалт үргэлж үнэлэгдэнэ! Энэ нь дараахыг агуулна:
- README файлууд
- Wiki хуудсууд
- Код сэтгэгдэл
- Сургамж ба гарын авлагууд
- Бусад хэлэнд орчуулга

## Хөгжүүлэлтийн Тохиргоо

### Урьдчилсан Нөхцөл

- Python 3.8 эсвэл түүнээс дээш хувилбар
- Git
- Root/Администраторын эрх (пакет барихын тулд)

### Орчныг Тохируулах

```bash
# Өөрийн fork-ийг татаж авах
git clone https://github.com/YOUR_USERNAME/AlbionInsight.git
cd AlbionInsight

# Виртуал орчин үүсгэх
python3 -m venv venv

# Виртуал орчинг идэвхжүүлэх
# Linux/macOS дээр:
source venv/bin/activate
# Windows дээр:
venv\Scripts\activate

# Хамаарлуудыг суулгах
pip install -r requirements.txt

# Хөгжүүлэлтийн хамаарлуудыг суулгах
pip install pylint flake8 black pytest
```

### Програмыг Ажиллуулах

```bash
# Linux/macOS дээр:
sudo venv/bin/python3 albion_insight.py

# Windows дээр (Администратороор):
python albion_insight.py
```

## Кодчилолын Стандартууд

Бид Python кодын хувьд PEP 8 стилийн удирдамжийг дагаж мөрдөнө. Таны код дараахыг дагаж байгаа эсэхийг шалгана уу:

- Зайд 4 сул зай ашиглах (табуляц биш)
- Хамгийн дээд мөрийн урт 100 тэмдэгт
- Утга санаа сайтай хувьсагч ба функцийн нэрүүд ашиглах
- Бүх функц ба классуудад docstring нэмэх
- Хэрэгтэй үед төрлийн сануулга оруулах
- Функцийг төвлөрөл ба товч байлгах

**Туслах хэрэгслүүд:**
```bash
# Black-ээр кодыг форматлах
black albion_insight.py

# Стилийн асуудлыг шалгах
flake8 albion_insight.py

# Linter ажиллуулах
pylint albion_insight.py
```

## Commit-ийн Мессежүүд

Тодорхой ба утга санаа сайтай commit мессежүүд бичнэ:

- Өнгөрсөн цаг ашиглах ("Added feature" биш "Add feature")
- Заалгалын үйлдэл ашиглах ("Move cursor to..." биш "Moves cursor to...")
- Эхний мөрийг 72 тэмдэгтээр хязгаарлах
- Холбогдох асуудал ба pull request-уудыг дурдах

**Жишээнүүд:**
```
Add damage meter export functionality

Fix network packet parsing for IPv6 connections

Update README with macOS installation instructions

Closes #123
```

## Pull Request-ийн Процесс

1. **Баримт бичгийг шинэчлэх** функцийн аливаа өөрчлөлтийн хувьд
2. **Шинэ функц эсвэл алдаа засалтын хувьд тест нэмэх**
3. **Илгээхээс өмнө бүх тест амжилттай болсныг баталгаажуулах**
4. **README.md-ийг шинэчлэх** шаардлагатай бол
5. **PR загварыг бүрэн дүүргэх**
6. **Холбогдох асуудлуудыг** PR тайлбарт холбох
7. **Хянагчдаас үзүүлэлт хүсэх**
8. **Санал нэхэмжлэлд хариу өгөх** шуурхай ба мэргэжлийн байдлаар

### PR Шалгалтын Жагсаалт

PR илгээхээс өмнө дараахыг баталгаажуулна уу:
- [ ] Код төслийн стилийн удирдамжийг дагаж байна
- [ ] Өөрийн үзүүлэлтийг дүүргэсэн
- [ ] Нарийн төвөгтэй код хэсгүүдэд сэтгэгдэл нэмсэн
- [ ] Баримт бичгийг шинэчилсэн
- [ ] Шинэ сөрөг сүрчилгээ үүсээгүй
- [ ] Тест нэмсэн ба амжилттай болсон
- [ ] Хамааралтай өөрчлөлтүүд нэгдсэн

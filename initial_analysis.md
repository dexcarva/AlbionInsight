# Análise Inicial do Projeto\n\nO projeto original, [AlbionOnline-StatisticsAnalysis](https://github.com/Triky313/AlbionOnline-StatisticsAnalysis), é uma ferramenta em C#/.NET, exclusiva para Windows, que oferece diversas funcionalidades como logger de loot, medidor de dano, etc. A motivação para o port em Python, o [AlbionInsight](https://github.com/dexcarva/AlbionInsight), é clara: fornecer uma solução multiplataforma (Linux, Mac, Windows) e aproveitar a flexibilidade do ecossistema Python.\n\nO repositório AlbionInsight já possui uma estrutura de documentação robusta, com múltiplos arquivos README e CONTRIBUTING em diversas línguas, o que facilita a próxima fase de gestão de Issues e Pull Requests.

# एल्बियन इनसाइट (Albion Insight)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**एल्बियन इनसाइट (Albion Insight)** गेम एल्बियन ऑनलाइन के लिए एक क्रॉस-प्लेटफ़ॉर्म (लिनक्स, विंडोज, मैकओएस) सांख्यिकी विश्लेषण उपकरण है, जिसे **फ्लेट (Flet)** फ्रेमवर्क का उपयोग करके **पायथन (Python)** में फिर से लागू किया गया है। इसे नेटवर्क ट्रैफ़िक का विश्लेषण करके वास्तविक समय में इन-गेम सांख्यिकी, जिसमें सिल्वर, फेम और कॉम्बैट डेटा (डैमेज मीटर) शामिल है, को ट्रैक करने के लिए डिज़ाइन किया गया है।

यह परियोजना मूल C#/WPF-आधारित `AlbionOnline-StatisticsAnalysis` उपकरण का एक आधुनिक, ओपन-सोर्स विकल्प है, जो मल्टी-प्लेटफ़ॉर्म संगतता और उपयोग में आसानी पर केंद्रित है।

## विशेषताएँ (Features)

*   **क्रॉस-प्लेटफ़ॉर्म संगतता:** लिनक्स, विंडोज और मैकओएस पर मूल रूप से चलता है।
*   **वास्तविक समय ट्रैकिंग:** एल्बियन ऑनलाइन पोर्ट्स (5055, 5056, 5058) पर यूडीपी पैकेट को सूंघने के लिए `Scapy` लाइब्रेरी का उपयोग करता है।
*   **डैमेज मीटर संरचना:** लाइव कॉम्बैट सांख्यिकी (किया गया डैमेज, की गई हीलिंग, डीपीएस) प्रदर्शित करने के लिए आवश्यक डेटा संरचनाएं और यूआई शामिल हैं।
*   **आधुनिक यूआई:** फ्लेट (Flet) के साथ निर्मित, एक तेज़, देशी दिखने वाला डेस्कटॉप एप्लिकेशन प्रदान करता है।
*   **सत्र प्रबंधन:** सत्र सांख्यिकी को शुरू करने, रोकने, रीसेट करने और सहेजने की अनुमति देता है।

## पूर्व-आवश्यकताएँ (Prerequisites)

*   पायथन 3.8+
*   **फ्लेट (Flet)** और **स्कैपी (Scapy)** लाइब्रेरी।
*   **रूट/प्रशासक विशेषाधिकार:** नेटवर्क पैकेट कैप्चर के लिए आवश्यक।

## इंस्टॉलेशन और सेटअप (Installation and Setup)

### विकल्प 1: त्वरित इंस्टॉलेशन (लिनक्स - अनुशंसित)

लिनक्स उपयोगकर्ताओं के लिए, हम स्वचालित इंस्टॉलेशन स्क्रिप्ट प्रदान करते हैं:

\`\`\`bash
# 1. रिपॉजिटरी क्लोन करें
git clone https://github.com/dexcarva/AlbionInsight.git
cd AlbionInsight

# 2. इंस्टॉलेशन स्क्रिप्ट चलाएँ
./install.sh

# 3. एप्लिकेशन चलाएँ
./run.sh
\`\`\`

`install.sh` स्क्रिप्ट निम्न कार्य करेगी:
- सिस्टम निर्भरताएँ स्थापित करें (`libpcap-dev`, `python3-pip`, `python3-venv`)
- एक पायथन वर्चुअल वातावरण बनाएँ
- सभी आवश्यक पायथन पैकेज (Flet, Scapy) स्थापित करें

`run.sh` स्क्रिप्ट स्वचालित रूप से रूट विशेषाधिकारों का अनुरोध करेगी और एप्लिकेशन चलाएगी।

### विकल्प 2: मैन्युअल इंस्टॉलेशन

#### 1. सिस्टम निर्भरताएँ स्थापित करें

**लिनक्स पर (डेबियन/उबंटू):**

\`\`\`bash
sudo apt update
sudo apt install libpcap-dev python3-pip python3-venv
\`\`\`

**विंडोज पर:**

[python.org](https://www.python.org/downloads/) से पायथन 3.8+ स्थापित करें

#### 2. पायथन निर्भरताएँ स्थापित करें

**लिनक्स पर (वर्चुअल वातावरण का उपयोग करके - अनुशंसित):**

\`\`\`bash
# वर्चुअल वातावरण बनाएँ
python3 -m venv venv

# वर्चुअल वातावरण सक्रिय करें
source venv/bin/activate

# निर्भरताएँ स्थापित करें
pip install flet scapy
\`\`\`

**लिनक्स पर (सिस्टम-व्यापी इंस्टॉलेशन):**

\`\`\`bash
pip3 install flet scapy --break-system-packages
\`\`\`

**विंडोज पर:**

\`\`\`bash
pip install flet scapy
\`\`\`

#### 3. एप्लिकेशन चलाना

चूंकि नेटवर्क स्निफिंग के लिए उन्नत विशेषाधिकारों की आवश्यकता होती है, इसलिए आपको एप्लिकेशन को रूट या प्रशासक के रूप में चलाना होगा।

**लिनक्स पर (वर्चुअल वातावरण के साथ):**

\`\`\`bash
sudo venv/bin/python3 -m albion_insight
\`\`\`

**लिनक्स पर (सिस्टम-व्यापी इंस्टॉलेशन):**

\`\`\`bash
sudo python3 -m albion_insight
\`\`\`

**विंडोज पर (कमांड प्रॉम्प्ट/पावरशेल को प्रशासक के रूप में चलाएँ):**

\`\`\`bash
python -m albion_insight
\`\`\`

एप्लिकेशन एक देशी डेस्कटॉप विंडो में खुलेगा।

## निष्पादन योग्य कैसे बनाएँ (How to Build an Executable)

एप्लिकेशन को **PyInstaller** का उपयोग करके एक स्टैंडअलोन निष्पादन योग्य में पैक किया जा सकता है। यह उपयोगकर्ताओं को पायथन या उसकी निर्भरताओं को स्थापित किए बिना एप्लिकेशन चलाने की अनुमति देता है।

लिनक्स, विंडोज और मैकओएस के लिए निष्पादन योग्य बनाने के विस्तृत निर्देशों के लिए, **[PACKAGING.md](PACKAGING.md)** गाइड देखें।

### त्वरित निर्माण (लिनक्स)

\`\`\`bash
source venv/bin/activate
pip install pyinstaller
pyinstaller --name "AlbionInsight" --onefile --windowed albion_insight/main.py
\`\`\`

निष्पादन योग्य `dist/` फ़ोल्डर में स्थित होगा।

## वर्तमान स्थिति (वास्तविक समय डेटा)

एप्लिकेशन में अब मूल C# प्रोजेक्ट से अनुवादित **फोटॉन प्रोटोकॉल डिकोडिंग (Photon Protocol Decoding)** तर्क शामिल है। यह एप्लिकेशन को नेटवर्क ट्रैफ़िक से सीधे `UpdateMoney`, `UpdateFame`, `KilledPlayer`, और `Died` जैसे वास्तविक समय की घटनाओं को संसाधित करने की अनुमति देता है।

**ध्यान दें:** प्रत्येक व्यक्तिगत कॉम्बैट इवेंट (जैसे `CastHit`, `Attack`) का पूर्ण अनुवाद एक सतत प्रयास है। वर्तमान कार्यान्वयन मुख्य सांख्यिकी और डैमेज मीटर के लिए संरचना पर केंद्रित है। डैमेज मीटर की डीपीएस गणना डिकोड की गई घटनाओं पर आधारित है।

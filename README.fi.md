# Albion Insight (Suomi)

Albion Insight on alustariippumaton (Linux, Windows, macOS) tilastoanalyysityökalu Albion Online -peliin, joka on toteutettu Pythonilla käyttäen Flet-kehystä. Se on suunniteltu seuraamaan reaaliaikaisia pelin sisäisiä tilastoja, mukaan lukien hopea, maine ja taistelutiedot (Damage Meter), analysoimalla verkkoliikennettä.

Tämä projekti on Python-porttaus alkuperäisestä Windows-työkalusta [AlbionOnline-Statistic](https://github.com/Triky313/AlbionOnline-Statistic).

## Ominaisuudet

*   **Monialustainen tuki:** Toimii Linuxissa, Windowsissa ja macOS:ssä.
*   **Reaaliaikainen seuranta:** Seuraa hopean, maineen ja taistelutilastoja.
*   **Damage Meter:** Tarkka mittari taistelun suorituskyvylle.
*   **Verkkoliikenteen analyysi:** Käyttää passiivista verkkoliikenteen sieppausta.

# Panduan Kontribusi untuk Albion Insight (Bahasa Indonesia)

Kami sangat menghargai kontribusi Anda untuk proyek Albion Insight!

Dengan mengikuti panduan ini, Anda dapat membantu kami menjaga proyek tetap terorganisir dan efisien.

## Cara Berkontribusi

### 1. Melaporkan Bug

Jika Anda menemukan bug, silakan laporkan melalui [Issues](https://github.com/dexcarva/AlbionInsight/issues).

*   **Sebelum Melapor:** Cari di antara isu yang sudah ada untuk memastikan bug belum dilaporkan.
*   **Detail yang Diperlukan:**
    *   Deskripsi yang jelas dan ringkas tentang bug.
    *   Langkah-langkah untuk mereproduksi perilaku tersebut.
    *   Sistem Operasi dan versi Python yang Anda gunakan.
    *   Teks kesalahan lengkap (jika ada).

### 2. Mengusulkan Fitur

Kami menyambut ide-ide baru!

*   Buka [Issues](https://github.com/dexcarva/AlbionInsight/issues) dan gunakan label `enhancement`.
*   Jelaskan fitur yang Anda usulkan dan mengapa itu akan bermanfaat bagi proyek.

### 3. Mengirimkan Pull Request (PR)

1.  **Fork** repositori ini.
2.  Buat **branch** baru untuk fitur atau perbaikan Anda (`git checkout -b fitur/nama-fitur` atau `git checkout -b perbaikan/nama-perbaikan`).
3.  Lakukan perubahan Anda.
4.  Pastikan kode Anda mengikuti standar gaya Python (gunakan `mypy` dan `flake8` jika memungkinkan).
5.  **Commit** perubahan Anda dengan pesan yang jelas dan deskriptif (lihat `CHANGELOG.md` untuk contoh).
6.  **Push** branch Anda ke fork Anda.
7.  Buka **Pull Request** ke branch `master` dari repositori utama.

Terima kasih atas waktu dan usaha Anda!
